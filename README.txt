# How the Game Works
Every time you lose the ball, your paddle shrinks,  
First person to have their paddle reduced to nothing loses

# Controls
Left Paddle - Up: W, Down: S  
Right Paddle - Up: Up Arrow, Down: Down Arrow  
  
Start Game: A  
Pause Game: Esc  
Activate Right Paddle Bot: B
Reset: R
